# Airline Reservation System

## Overview

The Airline Reservation System is a full-stack web application that enables users to search for available flights and book tickets through an interactive web interface. The application consists of a responsive frontend built with HTML, CSS, and JavaScript, and a backend developed using Spring Boot and MySQL.

## Features

### User Features

* Search available flights by source and destination
* View flight details including airline, schedule, and fare
* Book flight tickets
* View booking confirmation details
* Responsive and user-friendly interface

### Admin Features

* Add new flights
* Update flight information
* Delete existing flights
* Manage flight records

## Project Architecture

The application follows a layered architecture:

* **Controller Layer** – Handles HTTP requests and responses.
* **Service Layer** – Contains business logic.
* **Repository Layer** – Manages database operations using JPA.
* **Database Layer** – Stores flight and booking information.
* **Frontend Layer** – Provides user interaction through web pages.

## How to Run the Project

### Backend

1. Clone the repository.
2. Configure MySQL database credentials in `application.properties`.
3. Run the Spring Boot application.
4. Verify the backend is running successfully.

### Frontend

1. Open the frontend folder.
2. Launch `index.html` in a browser.
3. Ensure the backend server is running.
4. Search and book flights through the web interface.
